LiveTube-Coding — Referencia de funciones

Ver documentación en inglés

Introducción

LiveTube-Coding permite programar en vivo una rejilla de videos de YouTube desde un editor de código. Puedes usar tanto los nombres originales en inglés como una capa completa de alias en español.

Flujo típico: crear rejilla → buscar un video → usar su ID o etiqueta → controlar reproducción, audio, bucles y efectos visuales.

Ejemplo — sesión mínima
// 1. Crear rejilla 2×2 con un video inicial
crear(2, 2, "dQw4w9WgXcQ")

// 2. Buscar otro video y hacer clic en el resultado para insertar su ID
buscar("música experimental")

// 3. Sustituir el video de la celda 0
cambiar(0, "ID_QUE_INSERTASTE")

// 4. Reproducir todos con fade-in de 3 segundos
fundirEntrada(todos, 3)
reproducir(todos)

Atajos del editor

AtajoAcción
Shift+EnterEjecuta la línea actual o el texto seleccionado.
Alt+EnterAbre la ayuda en inglés de la función en esa línea.
Alt+Shift+EnterAbre la ayuda en español de la función en esa línea (panel derecho).
EscAlterna entre editor y rejilla. Si hay celdas seleccionadas, inserta sus índices en el editor.
Clic en celdaSelecciona/deselecciona una celda (overlay rojo). Útil con Esc para obtener índices.

Cómo seleccionar videos

Casi todas las funciones reciben como primer argumento qué videos controlar. Los índices empiezan en 0 y avanzan fila por fila, de izquierda a derecha.
Rejilla 3×3 — mapa de índices: ┌───┬───┬───┐ │ 0 │ 1 │ 2 │ ├───┼───┼───┤ │ 3 │ 4 │ 5 │ ├───┼───┼───┤ │ 6 │ 7 │ 8 │ └───┴───┴───┘
FormaTipoDescripción
0enteroUn solo video por índice.
all o nullallTodos los videos de la rejilla.
[0, 2, 5]ArrayLista de índices concretos.
not(3, 7)funciónTodos excepto los índices indicados. Ver not().
">3"stringExpresión evaluada por índice: ">3", "%2==0", etc.
Ejemplos de selección
reproducir(0)              // solo el primer video
pausar([1, 3])             // videos 1 y 3
silenciar(todos, true)     // silenciar todos
desenfocar(excepto(4), 3)  // desenfocar todos menos el 4

Variables globales útiles

NombreDescripción
randAlias de Math.random — números aleatorios.
iterAlias de setInterval — temporizadores personalizados.
allReferencia para seleccionar todos los videos (equivalente a null en muchas funciones).

create(filas, columnas, id)

Crea una rejilla nueva desde cero. Destruye la rejilla anterior. Filas y columnas deben ser divisores de 12: 1, 2, 3, 4, 6 o 12.
Parámetros
filasenteroNúmero de filas.
columnasenteroNúmero de columnas.
idstringID de YouTube (11 caracteres) o etiqueta definida con etiqueta() (p. ej. "Bolaño").
 
create(2, 2, "FGBhQbmPwH8")   // rejilla 2×2
create(3, 4, "dQw4w9WgXcQ")   // rejilla 3×4 (12 celdas)

createGrid(filas, columnas, id)

Alias de create(). Misma sintaxis; nombre más descriptivo usado en los ejemplos.
createGrid(3, 3, "FGBhQbmPwH8")

add(filasExtra, columnasExtra, id)

Amplía la rejilla existente añadiendo filas y/o columnas. El total resultante también debe cumplir la regla de divisores de 12.
Parámetros
filasExtraenteroFilas a añadir.
columnasExtraenteroColumnas a añadir.
idstringID de YouTube para celdas nuevas (opcional).
 
createGrid(2, 2, "abc123")
add(1, 0, "xyz789")   // pasa a 3×2; nuevas celdas cargan "xyz789"

search(consulta, opciones)

Busca videos en YouTube y abre un panel de resultados a la derecha del editor. Cada tarjeta muestra miniatura, título, canal, fecha e ID del video.

Clic en un resultado → inserta el ID entre comillas en el cursor del editor (listo para usar en cue() o createGrid()).

El campo Filtrar resultados… del panel reduce la lista en tiempo real sin volver a llamar a la API.

Parámetros
consultastringTexto a buscar (título, tema, artista, etc.).
opcionesnúmero | objetoOpcional. Número = cantidad máxima de resultados, o objeto con opciones avanzadas.
Opciones del objeto
maxResultsenteroResultados a pedir (1–50, por defecto 20).
orderstringrelevance (defecto), date, viewCount, rating, title.
Ejemplos de búsqueda
// Búsqueda simple
search("música electroacústica")

// Limitar a 10 resultados
search("John Cage prepared piano", 10)

// Más recientes primero
search("performance art", { maxResults: 15, order: "date" })

// Flujo completo: buscar → clic en resultado → usar ID
search("Sang Won Lee aural cavity")
// (clic en un video del panel)
cue(0, "ID_INSERTADO")
play(0)
Nota: La búsqueda requiere una clave válida de YouTube Data API v3 en js/config.local.js (copia desde js/config.example.js; no se sube a Git). Solo se listan videos embeddables. Si falla, revisa la consola y las restricciones de la clave en Google Cloud.

cue(videos, id, cancelarBucle)

Sustituye el contenido de las celdas seleccionadas por otro video de YouTube. Reinicia velocidad y volumen; por defecto cancela bucles activos.
Parámetros
videosselecciónCeldas a actualizar.
idstringNuevo ID de YouTube.
cancelarBuclebooleanotrue (defecto) detiene loop/loopAt en esas celdas.
 
cue(0, "FGBhQbmPwH8")
cue([2, 3], "nuevoId", false)   // mantiene bucles activos

Reproducción

play(videos)

Inicia la reproducción de los videos seleccionados.
play(all)
play([0, 1])

pause(videos)

Pausa los videos seleccionados.
pause(all)
pause(3)

seek(videos, segundos)

Salta a un instante concreto del video, en segundos (admite decimales).
seek(all, 144)        // todos al segundo 144
seek([2, 3], 90.5)    // dos celdas a 90.5 s

speed(videos, velocidad)

Cambia la velocidad de reproducción. 1 = normal; 0.5 = mitad; 2 = doble.
speed(all, 1.25)
speed(0, 0.5)
speed([1, 2], 2)

sync(videos, índice)

Alinea los videos seleccionados al tiempo actual del video en índice.
sync(all, 0)   // todos siguen al video 0

setQ(videos, calidad)

Establece la calidad de reproducción de celdas concretas.

Valores: small, medium, large, hd720, hd1080, highres.

setQ([0, 1], "hd720")

setQoverall(calidad)

Define la calidad por defecto para videos que se carguen después.
setQoverall("medium")

Audio

mute(videos, silenciar)

true silencia; false restaura el sonido.
mute(all, true)
mute(0, false)

volume(videos, vol)

Fija el volumen entre 0 (mudo) y 100 (máximo). También ajusta la opacidad del overlay de la celda.
volume(0, 50)
volume(all, 80)

turnup(videos, diff)

Incrementa o decrementa el volumen actual. Usa valores negativos para bajar.
turnup(all, 10)      // +10
turnup([1, 2], -20)  // -20

fadeIn(videos, duración)

Sube el volumen de forma gradual. duración en segundos (por defecto 5).
fadeIn([0, 1], 3)   // fade-in de 3 segundos

fadeOut(videos, duración)

Baja el volumen de forma gradual. duración en segundos (por defecto 5).
fadeOut(all, 5)

Tiempo, bucles y ritmo

loop(videos, atrás, intervalo, fase)

Crea un bucle relativo: retrocede atrás segundos desde el momento actual y repite cada intervalo segundos. fase desfasa el intervalo de cada celda para crear patrones rítmicos. 
loop(0, 2, 4)           // celda 0: bucle de 2 s cada 4 s
loop(all, 1, 3, 0.2)    // todas con fase escalonada

loopAt(videos, enTiempo, intervalo, fase)

Bucle desde un instante absoluto enTiempo (segundos), repitiendo cada intervalo.
loopAt(3, 197, 4)
loopAt([0, 3], 152, 0.2)
loopAt(2, 164.57, 0.2)

unloop(videos)

Detiene todos los bucles activos en las celdas indicadas.
unloop(all)
unloop([1, 2])

here(videos)

Primera llamada: guarda el instante actual como marca.
Segunda llamada: crea un bucle entre la marca y el momento actual (equivalente a loopAt con ese rango). 
here(0)    // marca
// ... deja pasar un fragmento ...
here(0)    // buclea entre marca y ahora

phase(videos, intervalo)

Desfasa en el tiempo los videos seleccionados. Intervalo positivo o negativo cambia la dirección del desfase.
phase(all, 0.5)

delay(videos, intervalo)

Retrasa el inicio de reproducción entre celdas. Las celdas posteriores arrancan más tarde.
delay(all, 1)   // cada celda empieza 1 s después

jump(videos, num, fase)

Salta num segundos (positivo = adelante, negativo = atrás). Con fase, los saltos se escalonan en el tiempo.
jump(all, 10)
jump([0, 1], -5, 0.5)

not(índices…)

Devuelve un array con todos los índices de la rejilla excepto los indicados. Se usa dentro de otras funciones.
// Rejilla 3×3:
not(1)       // [0,2,3,4,5,6,7,8]
not(8, 7)    // todos menos 8 y 7

blurVideo(not(3), 4)   // desenfocar todos excepto celda 3

Efectos visuales

Los efectos usan filtros CSS en GPU sobre el iframe y, para scanlines, viñeta o grano, capas overlay. No requieren shaders WebGL (el iframe de YouTube no permite leer sus píxeles). Para quitar todo: limpiarFx(todos).

grayscale(videos, activar)

Activa (true) o desactiva (false) escala de grises.
grayscale([0, 2, 4, 6], true)
grayscale(all, false)   // restaurar color

adjustLight(videos, brillo, contraste)

1 = valor normal en ambos. Menor brillo = más oscuro; mayor contraste = más definición.
adjustLight(all, 0.7, 1.3)

hueRotate(videos, grados)

Rota el matiz del color en grados (0–360).
hueRotate(2, 90)

blurVideo(videos, cantidad)

Aplica desenfoque gaussiano en píxeles.
blurVideo(not(3), 4)   // todo menos celda 3

saturate(videos, cantidad)

1 = normal; 0 = sin color; 2 = muy saturado.
saturate(all, 1.8)

setFilter(videos, filtroCSS)

Aplica cualquier cadena de filtro CSS válida.
setFilter(0, "contrast(2) brightness(0.8)")
setFilter(null, "none")   // quitar filtros de todos

animateFilter(videos, filtroCSS, duración)

Transición suave hacia un filtro CSS. Duración en segundos (por defecto 1.5).
animateFilter([1, 3, 5], "hue-rotate(180deg) saturate(2) blur(1px)", 3)

tiltVideo(videos, rotateX, rotateY)

Inclinación 3D del video. Valores en grados para rotación. No cambia la escala; usa scale().
tiltVideo(2, 10, -10)
tiltVideo(null, 0, 0)   // quitar inclinación

scale(videos, escalaX, escalaY)

Escala del video. 1 = tamaño normal, 0.5 = mitad, 2 = doble. Si omites escalaY, la escala es uniforme.
scale(all, 0.5)
scale(2, 1.5)
scale(null, 1)   // restaurar escala

text / texto(videos, contenido, opciones)

Coloca texto sobre celdas. Opciones: font, size, color, weight, align, x, y (0–1 dentro de la celda). También: argumentos cortos texto(0, "hola", 48, "#63b3ed") o marcas inline [[color:#63b3ed|size:56]]texto[[/]].
texto(0, "LiveTube", { size: 56, color: "#63b3ed", font: "Inter" })
texto(0, "[[color:#f56565|size:80]]AO[[/]] [[color:#63b3ed]]vivo[[/]]", 42)
textoFuente(0, "JetBrains Mono")
textoTamano(0, 72)
textoColor(0, "#f56565")
textoEstilo(0, { align: "left", x: 0.1, y: 0.2 })

commandText / comando(contenido, opciones)

Texto sobre el editor de comandos (donde escribes el código). Acepta las mismas opciones y marcas inline que texto().
comando("[[size:28|color:#63b3ed]]LiveTube[[/]] en vivo")
estiloComando({ font: "Inter", size: 22 })
estiloEditor({ font: "JetBrains Mono", size: 18, color: "#fbd38d" })
efectoComando("caer", 2)
efectoEditor("ofuscar", 3)   // efecto sobre el código del editor
efectoEditor("normal")         // = restaurarEditor()
restaurarEditor()              // atajo: Esc, clic, Ctrl+Shift+E
limpiarComando()

textEffect / textoEfecto(videos, efecto, duración)

Efectos: caer / drop (letras caen al suelo), colapsar / collapse, ofuscar / obfuscate. normal restaura el texto.
textoEfecto(0, "caer", 2.5)
textoEfecto(todos, "colapsar", 1)
textoEfecto(0, "ofuscar", 3)
textoEfecto(0, "normal")
textoInteractivo(0, function () { textoColor(0, "#68d391") })
limpiarTexto(0)

opacity(videos, alpha)

Opacidad visual: 0 invisible, 1 opaco.
opacity(3, 1)       // foco en celda 3
opacity(null, 1)    // restaurar todos

applyPreset(videos, nombre, animar)

Aplica un estilo predefinido. Si animar es true (defecto), la transición es suave. Ver también presetOverlay().
PresetEfecto
calmSuave, gris parcial, luminoso.
tenseAlto contraste y saturación.
dreamBorroso, brillante, matiz desplazado.
chaosColores invertidos, saturado, leve blur.
darkOscuro con contraste.
softSuave, ligeramente desenfocado.
retroSepia y contraste vintage.
neonColores neón saturados.
horrorGris, oscuro, alto contraste.
vhsLook cinta VHS (solo filtro).
noirCine negro.
cyberpunkCian/magenta saturado.
bolanoSepia suave (preset temático).
 
aplicarPreset(1, "tense")
aplicarPreset(todos, "bolano", true)
listarPresets()   // ver todos en consola

applyOverlayPreset / presetOverlay(videos, nombre)

Combina filtro CSS con capas overlay (scanlines, viñeta, grano, glitch).
PresetIncluye
vhsFiltro VHS + líneas + viñeta + grano.
crtScanlines CRT + viñeta.
glitchGlitch animado + contraste alto.
 
presetOverlay(todos, "vhs")
presetOverlay(0, "crt")

sepia(videos, cantidad)

1 = sepia completo. Alias en español: mismo nombre.

invert / invertir(videos, cantidad)

Invierte colores (1 = negativo completo).

duotone / duotono(videos, tonoGrados)

Efecto de dos tonos. Por defecto 200 grados de matiz.
duotono(todos, 220)

mixBlend / mezcla(videos, modo)

Modo de mezcla CSS: screen, multiply, difference, etc.

flip / espejo(videos, horizontal, vertical)

espejo(todos) voltea horizontalmente.

vignette / vigneta(videos, fuerza)

Viñeta oscura en los bordes. fuerza entre 0 y 1 (defecto 0.75).

scanlines / lineas(videos, opacidad)

Líneas horizontales estilo monitor CRT.

filmGrain / grano(videos, opacidad)

Grano de película (ruido SVG).

glitch(videos, activar)

Activa (true) o desactiva glitch animado.

pulse / pulso(videos, activar)

Pulso rítmico de brillo.

pixelate / pixelar(videos, bloques)

Pixelación simulada. bloques = factor de escala (defecto 6).

resetFx / limpiarFx(videos)

Elimina filtros, overlays, animaciones y restaura el tamaño del reproductor.
limpiarFx(todos)

listarPresets()

Muestra en la consola los nombres de VisualPresets y OverlayPresets.

Etiquetas para IDs de video

En lugar de recordar IDs largos de YouTube (p. ej. iqtYrcE27hE), puedes definir etiquetas con nombres cortos y usarlas en crear(), agregar() y cambiar().

Las etiquetas no distinguen mayúsculas: "Bolaño", "bolaño" y "BOLAÑO" son equivalentes.

etiqueta / etiquetar / alias(nombre, id)

Registra una etiqueta. El segundo argumento puede ser un ID de YouTube o el nombre de otra etiqueta ya definida.
Parámetros
nombrestringNombre corto, p. ej. "Bolaño", "Rim".
idstringID de YouTube (11 caracteres) u otra etiqueta.
Definir y usar etiquetas
// Definir una etiqueta nueva
etiqueta("Rim", "dQw4w9WgXcQ")
etiqueta("Bolaño", "iqtYrcE27hE")

// Usar la etiqueta en lugar del ID
crear(1, 1, "Bolaño")
cambiar(todos, "Rim")

// Apuntar una etiqueta a otra
etiqueta("Favorito", "Bolaño")
crear(2, 2, "Favorito")

listarEtiquetas()

Muestra una tabla en la consola del navegador (F12) con todas las etiquetas y sus IDs. También devuelve un objeto con el mapa completo.
listarEtiquetas()
Etiqueta incluida por defecto
EtiquetaID de YouTube
Bolaño / BolanoiqtYrcE27hE
 
crear(1, 1, "Bolaño")   // listo para usar sin configurar nada

Dónde funciona la resolución de etiquetas: crear(), createGrid(), agregar(), cambiar() / cue().

Para añadir etiquetas fijas al proyecto, edita el objeto videoAliases al inicio de js/main.js.
